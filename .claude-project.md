# Nexa — Site notes

## Dev server
Run with `npm run dev` (package.json already sets NODE_ENV=development).
The environment has NODE_ENV=production set globally — the dev script overrides it.

## Spline 3D
Hero uses a CSS/Framer Motion animated orb (`components/HeroOrb.tsx`) as a placeholder.
Replace with a real Spline scene: install `@splinetool/react-spline` (already in deps),
then swap `HeroOrb` with a dynamic import of `Spline` + your scene URL.

## Language toggle
Managed in `context/LanguageContext.tsx`. All strings are in `lib/translations.ts`.

## Contact form
`components/Contact.tsx` — submit handler is a placeholder (`setSubmitted(true)`).
Wire up to Formspree, Resend, or any API route.
